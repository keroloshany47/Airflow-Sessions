#  Airflow Sessions

This repository contains **two sessions** explaining core concepts of **Apache Airflow** using practical examples and slides.

---

##  What I Covered

I explained and demonstrated the following Airflow concepts:

* DAG basics and task dependencies
* ETL pipelines (Extract, Transform, Load)
* Branching workflows using conditions
* Task grouping for better organization
* XCom for passing data between tasks
* Assets handling in Airflow workflows
* Custom operators and Python decorators in Airflow
